通过结构化的动手学习路径备考 Red Hat Certified System Administrator（RHCSA，EX200）考试。本路线图侧重 RHEL 10 上的实践型系统管理、性能型考试任务与真实企业场景，涵盖基础工具与 Shell 脚本、软件与运行系统、本地存储与文件系统、网络与服务，以及用户、组与安全。后续将逐步加入 RHCSA 课程、实验环境与模拟考试练习，帮助你建立与 EX200 目标一致的能力。

[在 LabEx 浏览项目](https://labex.io/zh/learn/rhcsa)

## 项目
