Red Hat Certified System Administrator（RHCSA、EX200）試験に向けた、体系的かつ実践的な学習パスです。RHEL 10 上の実務的なシステム管理、パフォーマンスベースのタスク、現実のエンタープライズシナリオに焦点を当てます。RHCSA コース、ラボ、模擬試験の演習は順次追加され、EX200 の目標に沿ったスキルを育成します。

[LabEx でプロジェクトを見る](https://labex.io/ja/learn/rhcsa)

## プロジェクト

初心者向けの 4 個の RHCSA トレーニング プロジェクトで、言語の重要な概念を小さく完成したプログラムに変えて学べます。各プロジェクトは LabEx のすぐ使える環境で開くため、構築に集中できます。
